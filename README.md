JavaScript kaverilista -projekti

Sivulla 1. on yksinkertaisempi versio, joka kysyy kymmenen kaverin nimen ja lisää ne listaan. Ohjelma ei anna lisätä tyhjää nimeä vaan kenttään on syötettävä nimi,
jotta ohjelma jatkuu.

Sivulla 2. on hieman monipuolisempi versio, jossa käyttäjä voi lisätä kavereita sekä myös poistaa ja järjestellä lisäämiään nimiä listassa.
Ohjelma ei anna lisätä tyhjää nimeä vaan huomauttaa siitä käyttäjää.
